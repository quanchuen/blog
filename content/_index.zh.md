# 您好！

我叫 Quan Zhou，喜爱平淡简单的生活。虽说喜爱，但更像是一种期望。这个网站随意的记录着一些无聊的笔记，有的是为了未来重现，有的就是觉得“干了一件厉害的事情，赶紧记下来”，还有的是一些抱怨（比如 docker 怎么突然开始用奇怪的 CIDR 了）。


这是我第 N 个博客了，因为经常一年一博，很多因为域名忘记续费，或者过了赎回期被抢注而~永远~消失了。虽然在邮箱的角落里还有一些备份，但是我并没有觉得有必要恢复以前的。
